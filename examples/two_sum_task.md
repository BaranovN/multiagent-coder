# Two Sum

Read two lines from stdin:

1. A target integer `T`.
2. A space-separated list of integers.

Print the **0-based indices** (space-separated, ascending) of the two
distinct list entries whose values sum to `T`.

If several valid pairs exist, print any one pair. If no valid pair exists,
print `-1`.

Constraints:

- 2 ≤ len(list) ≤ 200000
- values fit in a signed 64-bit integer
- the list may contain duplicates
